# Car Rental Management System

A Java web application developed using Servlets, JSP, JPA (Hibernate) and PostgreSQL.
The system allows admins to manage cars and customers to book available cars.

## Features
- Add new cars (Admin)
- View all cars
- View available cars
- Book a car
- Complete booking
- View all bookings

## Technologies Used
- Java
- Servlets
- JSP
- JPA (Hibernate)
- PostgreSQL
- Apache Tomcat
- Maven

## How to Run
1. Clone the repository
2. Import as Maven project
3. Configure PostgreSQL database
4. Update persistence.xml
5. Run on Apache Tomcat
